Welcome to django-chameleon's documentation!
============================================

**Description**:

This app will let you change the template/theme dinamically in your Django_ website.
This way the user can select the best theme that fits his likes, browser, resolution... in the website

**Features**:

- Session theme store (the theme sets until the cookie/session is deleted)
- No limit of themes
- Project example
